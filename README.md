# COMP3304 Project - Waiterly

## Project Overview
Waiterly is a digital system developed to support restaurant, café, and bar operations.
The system helps waiters manage customer orders efficiently and improves the overall service experience for both staff and customers.

## Objectives
- Increase service speed and efficiency
- Allow customers to easily view menu items and ingredients
- Allow customers to see the food allergens more easily
- Support waiters by organizing and managing orders
- Reduce operational costs for restaurant owners
