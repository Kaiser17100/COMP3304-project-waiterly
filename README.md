#COMP3304 PROJECT - WAITERLY
##PROJECT DESCRIPTION: 
Waiterly is a system designed to assist restaurant/café/bar waiter operations by managing orders and improving service efficiency.

##PROJECT GOALS
- improve service efficiency
- help customers to easily know the contents of their beverage/meal 
- assist waiters in workload 
- reduce expenses for owners of restaurants due to the decreased need for waiters
